Requisitos de la aplicación, tener Java instalado

Eclipse, Visual Studio Code con Angular, XAMPP

Desde XAMPP, crearemos una base de datos vacia, llamada login, vacia.

En Visual Studio Code, haremos un npm install, para instalar todos los modulos del proyecto y un ng serve -o
para abrirlo

En Eclipse, importaremos, abriremos el proyecto y (con Apache y MySQL abiertos) iniciaremos la aplicación
